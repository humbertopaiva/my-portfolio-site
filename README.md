## 🚀 Rodando o projeto na sua máquina

Para instalar o projeto em sua máquina, siga estas etapas:

1 - Clone o repositório para o seu computador

2 - Inicialize o repositório e execute o comando

```
yarn start 
```

ou

```
npm start 
```
